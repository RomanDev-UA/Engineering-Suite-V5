🌟 Описание проекта
Этот инструмент — мост между практическим опытом в сварке и цифровой инженерией. Он автоматизирует рутинные задачи мастера по металлу: расчет материалов, минимизацию отходов и оценку сварочных рисков.

✨ Основные функции
Умный расчет каркаса: Автоматическое определение количества стоек и перемычек на основе габаритов (L, W, H).
Алгоритм оптимизации раскроя: Программа вписывает детали в стандартные 6-метровые хлысты с учетом ширины реза (kerf), сводя остатки к минимуму.
Физико-экономический движок: Расчет общего веса (брутто/нетто), стоимости материалов и процента отхода.
Интеллектуальный помощник сварщика: Анализ толщины стенки металла и выдача рекомендаций (настройка тока, риск деформации).
3D Визуализация: Генерация модели будущей конструкции для предварительного просмотра.
Профессиональная отчетность: Автоматическое создание PDF-отчетов для клиентов или цеха.
